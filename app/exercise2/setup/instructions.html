Instructions

You are to implement a basic list controller & view. You will fetch any data required from the server.

You you may only use built-in javascript objects & AnuglarJS, bootstrap css - use of other libraries is not permitted. Some of the boilerplate has been filled in for you.

For the JS in the following tasks you are to use AngularJS controller as syntax. (info here)

Task 1 - Get a list of users (Function declared)

You will implement a controller function getUsers. The function will make a http GET request to '/users' endpoint. This end point will return a collection of 10 user objects.

The function will be invoked upon controller initialisation and will set a variable called list which will be made available to the view.

Note
  • Function must only use Angular's $http service or ngResource modules' $resource service to make the requests.
Task 2 - Display the list of users

The list of users will be displayed, sorted in ascending alphabetical order by their forenames.

Tips
  • Angular has a list of filters. Filters
Task 3 - Show an alert while list is loading.

While the front end is waiting for the collection of users to be returned - a message should be shown to inform the user.

 Task 4 - Let the user refresh the list random users.

There is a button in the view, use this button to refresh the list of users.

 Task 5 - Style view to match working example

Use bootstrap CSS & /app/exercise2/solution.css to style the elements to closely match the working example.

 Task 6 - User images will not display on small devices

Profile images will only be displayed when viewport width is greater than 480px.
