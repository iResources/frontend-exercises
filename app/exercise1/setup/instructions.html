Instructions

You are to implement a very basic shopping list controller & view.

You you may only use built-in javascript objects & AnuglarJS, bootstrap css - use of other libraries is not permitted. Some of the boilerplate has been filled in for you.

For the JS in the following tasks you are to use AngularJS scope syntax.

Task 1 - Add an item (Function declared)

You will implement a controller function called addItem. It add an item to the list of shopping items.

Note
  • Function should allow the addition of duplicate items (exact match).
  • Function should not add empty items.
 Task 2 - Remove an item (Function declared)

You will implement a controller function called removeItem. It add an item from the current list of shopping items.

Tips
  • There are few ways to do this, try to think collection based.
 Task 3 - Implement the view template

You will create make the view template. It will interact with the user with functions which are declared in the controller.

It will display the current list of items, allow the user to enter an item & remove an item and the view will refresh with the new list.

Items will be displayed in alphabetical order.

 Task 4 - Style the view

Style the shopping list & the inputs to look like the demo version.
